I love the idea of a dark and light theme on a website. After discovering this feature on the Next.js website, I decided to experiment with it myself.

To begin, I downloaded the 'next-themes' package.

```jsx
$ npm install next-themes
# or
$ yarn add next-themes
```

Then, I made a new file in the component

```jsx
import { useTheme } from "next-themes"
```

Then I made a theme component as follows:

```jsx
"use client"

import * as React from "react"
import { ThemeProvider as NextThemesProvider } from "next-themes"
import type { ThemeProviderProps } from "next-themes/dist/types"

export function ThemeProvider({ children, ...props }: ThemeProviderProps) {
  return <NextThemesProvider {...props}>{children}</NextThemesProvider>
}
```

This code establishes a custom 'ThemeProvider' component, serving as a wrapper around the 'ThemeProvider' provided by the 'next-themes' library. It is intended for use in the application's root to seamlessly manage themes across the entire component tree.
